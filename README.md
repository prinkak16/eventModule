# HOLA! React with Ruby Jumpstart

Ruby version: `3.2.0`

Rails version: `7.0.6`

Copy `.env.example` as `.env` and populate your details

Run following commands
```
bundle install
rails db:create
rails db:migrate
```

### To sync locatable:-
Run
```
bundle exec sidekiq
```

Goto: [Sync Locatable](http://localhost:3000/saral-locatable/states)
